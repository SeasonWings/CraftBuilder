# CraftBuilder Pro - 生产材料计算工具

CraftBuilder Pro 是一个专为复杂生产流程设计的材料计算工具。它支持多层级材料依赖管理、自动递归计算基础材料需求，并生成详细的分解报告。

## 核心功能

- **材料库管理**：支持三级材料分类（基础、中间、最终产品），支持动态增删改查及层级依赖配置。
- **递归计算引擎**：自动计算生产指定数量产品所需的所有一级基础材料总量。
- **层级分解报告**：提供交互式树状分解视图，清晰展示每层生产环节的消耗。
- **批量生产计算**：支持同时计算多种不同产品的合并基础材料需求。
- **本地持久化**：所有配置的材料数据自动保存至浏览器 LocalStorage。
- **响应式设计**：完美适配 PC 端和移动端访问。

## 技术栈

- **框架**: React 18 + Vite
- **语言**: TypeScript
- **状态管理**: Redux Toolkit (RTK)
- **UI 组件库**: Ant Design (antd) 5.x
- **图标**: Ant Design Icons & Lucide React
- **样式**: Tailwind CSS
- **测试**: Vitest + Testing Library

## 安装与运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 执行单元测试
```bash
npm run test
# 或者运行单次测试并生成报告
npx vitest run
```

### 4. 构建生产环境代码
```bash
npm run build
```

## 使用指南

1. **材料管理**：在“材料管理”页面，首先添加一级基础材料（如：铁矿、煤炭）。
2. **配置配方**：添加二级或三级材料时，选择其所需的前置原料及其数量。
3. **开始计算**：切换至“材料计算”页面，选择目标产品和数量，即可实时查看基础材料汇总和详细分解树。
4. **批量清单**：在计算页面下方，可添加多个不同产品，一键汇总所有生产任务的基础材料需求。
